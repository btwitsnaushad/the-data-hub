# AI Prompt Engineering Log

During the development of this REST API project, AI was used for concept clarification, debugging, and understanding backend implementation strategies. The following prompts were used during development:

1. How can I implement a custom global middleware in Express.js that logs the HTTP method, request URL, and current timestamp for every incoming request?

2. What is the best way to implement CRUD operations using an in-memory array in Express.js, especially for updating a specific object with a PUT request?

3. How should I implement GET `/posts/:id` so that it retrieves a single blog post based on the route parameter instead of returning a static response?

4. How can I create a mock `/login` endpoint in Express.js that returns a fake JWT token and can later be extended with real authentication?
